We then parallelize the processing of the database by running it on high-performance computing clusters. The results are serially filtered from the database based on the following criteria, in order: i) firing rate falls in the range of [4.8, 10]Hz at 50pA current injection; ii) firing rate falls in the range of [6.5, 13]Hz at 80pA current injection; iii) firing rate falls in the range of [7.5, 14.2]Hz at 100pA current injection; iv) quasi-linear f-I curve based on values from i-iii; v) the presence of a post-inhibitory rebound at a -80pA current injection; vi) the presence of spike rate adaptation. These criteria were extracted from the measured electrophysiological properties of tSPNs (Fig.1). tSPN models that satisfy all the six criteria above will be used to examine recruitment principles. 
Processing of the database will be parallelized with the MATLAB Parallel Computing Toolbox and run on the Emory High-performance Computing Cluster named TARDIS with 64 cores.
